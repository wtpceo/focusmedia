## 🔍 검색 기능

상단 검색창에서 **아파트명, 주소, 동 이름**으로 검색할 수 있습니다.

**예시:**
- `목동` → 목동 지역 아파트 검색
- `래미안` → 래미안 아파트 검색
- `강남구` → 강남구 소재 아파트 검색

---

## 🎛️ 필터 기능

우측 상단 **필터 버튼**을 클릭하면 필터 패널이 열립니다.

### 1. 매체 구분
원하는 매체만 선택하여 볼 수 있습니다.
- **전체**: 모든 매체 표시
- **포커스미디어**: 포커스미디어만 표시
- **타운보드 만첨**: 타운보드 만첨 단지만 표시
- **타운보드 가동**: 타운보드 가동 단지만 표시

> 💡 **다중 선택 가능**: 여러 매체를 동시에 선택할 수 있습니다.

### 2. 판매등급 (포커스미디어 전용)
- S+, S, A+, A, B 등급별 필터링

### 3. 건물유형
- 아파트, 오피스텔, 주상복합

### 4. 지역
- 구/군 단위로 필터링

### 5. 설치시기
- 연도별 필터링 (2020년~2025년)
- 미설치 항목 필터링

## 📋 아파트 정보 확인

마커를 클릭하면 하단에 정보 패널이 표시됩니다.

### 포커스미디어 정보
| 항목 | 설명 |
|------|------|
| 세대수 | 아파트 총 세대수 |
| 인구수 | 추정 거주 인구 |
| 층수 | 건물 층수 |
| 평형 | 대표 평형 |
| 수량 | 설치된 TV 수량 |
| 대당가 | TV 1대당 광고 단가 |
| 4주 금액 | 4주 광고 금액 |

### 타운보드 정보
| 항목 | 설명 |
|------|------|
| 세대수 | 아파트 총 세대수 |
| 평형 | 평형 정보 |
| 수량 | 가동 수량 |

> ⚠️ 타운보드는 대당가, 4주 금액 정보가 없습니다.

## 💰 광고 금액 상세 보기 (포커스미디어 전용)

**4주 금액** 부분을 클릭하면 기간별 할인 금액을 확인할 수 있습니다.

| 기간 | 할인율 | 계산 방식 |
|------|--------|-----------|
| 4주 | - | 기본 금액 |
| 24주 | 5% 할인 | 4주 × 6 × 0.95 |
| 48주 | 10% 할인 | 4주 × 12 × 0.90 |

> ※ 부가세 별도

## 🧭 편의 기능

### 주소 복사
정보 패널에서 **주소복사** 버튼을 클릭하면 주소가 클립보드에 복사됩니다.

### 길찾기
**길찾기** 버튼을 클릭하면 카카오맵에서 해당 위치로 길찾기가 실행됩니다.
- 모바일: 카카오맵 앱 실행
- PC: 카카오맵 웹 페이지 열림

### 현재 위치
우측 하단 **위치 버튼**을 클릭하면 현재 내 위치로 지도가 이동합니다.

### 화면 고정
**자물쇠 버튼**을 클릭하면 지도 이동이 고정됩니다.
- 마커 클릭 시 지도가 이동하지 않음
- 다시 클릭하면 고정 해제

### 주변 업체 검색
정보 패널에서 **주변 검색** 버튼을 클릭하면 해당 아파트 주변의 업체를 검색할 수 있습니다.
- 편의점, 카페, 음식점 등 카테고리별 검색
